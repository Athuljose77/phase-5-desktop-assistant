# Phase-5 — System Control Features

Your assistant now has **Full System Control**.

## 🎮 New Commands

| Command | Example | What it does |
|---|---|---|
| **Volume** | `set volume 50`, `mute`, `unmute` | Controls system audio |
| **Brightness** | `set brightness 80` | Adjusts screen brightness |
| **Screenshot** | `take screenshot` | Saves screen to `screenshots/` folder |
| **Lock** | `lock screen` | Locks your PC |
| **Open App** | `open spotify` | Searches Start Menu & launches app |
| **Close App** | `close notepad`, `quit excel` | Terminates the app |

## 🚀 How to Run
```bash
python main.py
```
